Category

কেন: প্রোডাক্টসবকে সংগঠিত ও ফিল্টার করার জন্য ক্যাটেগরি দরকার।

HTTP Path বিবরণ
GET /category সকল ক্যাটেগরি লিস্ট পেতে।
POST /category নতুন ক্যাটেগরি তৈরি। (multipart/form-data — ছবি সহ)
GET /category/:id একটি স্পেসিফিক ক্যাটেগরি দেখতে।
PATCH /category/:id ক্যাটেগরি আপডেট (formSupport middleware প্রয়োগ করা)।
DELETE /category/:id ক্যাটেগরি ডিলিট করা।

POST /category — Request (multipart/form-data)

Createt feald

{
  "name": "your category name",
  "file": "your category image" (optional),
  
}
Note: Slug ফিল্ড automatically generated হয়, আলাদা key প্রয়োজন নেই।

Category URL: https://eshop-production-f5b0.up.railway.app/api/v1/category

Middleware: categoryMulter (file upload), formSupport (form-data parse for patch)।
